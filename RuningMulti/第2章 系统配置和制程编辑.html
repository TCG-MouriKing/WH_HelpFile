# 第2章

软件主界面包含以下几个模块：

1）菜单栏；2）检测结果统计；3）当前制程名称；4）图像视窗区；5）参数设置区；6）检测缺陷统计；7）汇总统计；8）状态栏；

# 2.1工程新建与参数配置

在新建工程前需要对相机个通讯进行设置。

# 2.1.1相机设置

# 2.1.1.1相机搜索

1）点击打开菜单栏的‘相机与通讯’按钮，打开相机配置界面

2）进入相机界面后点击搜索按钮搜索已连接的相机

3）勾选需要使用的相机，并打钩退出

# 2.1.1.2相机搜索

1）添加相机后，确认每个相机都已处理连接状态，根据需要对相机参数进行修改，相机工位名称根据项目需要进行修改，方便区分相机，设置好相机参数后点击保存按钮保存配置参数

# 2.1.2通讯设置

# 2.1.2.1 添加默认通讯

1）点击打开菜单栏的‘相机与通讯’按钮，打开通讯配置界面

2）添加一个通讯

# 2.1.2.2 通讯IP设置

1）添加默认通讯后进入参数设置界面对通讯进行设置，首先设置IP地址和端口号，根据项目实际需求进行设置

# 2.1.2.3 受控相机设置

1）受控相机既是参与通讯绑定的相机，添加相机到受控相机中就可以通过通信控制相机，不添加既是不受通讯控制

打开：点开受控相机按钮设置相机

2）添加：将相机添加到成员列表中，根据项目需要配置受控相机

2）结果分组：结果分组是为每个相机分配结果汇总，处在同一组即视为将本组相机的检测结果归为同一个结果，可根据项目情况设置

# 2.1.2.4 触发拍照设置

触发拍照是相机接受通讯的命令设置，可设置接收触发信号，流水号，可编辑信号（舟号，漕号，管号，列号等，可自由编辑）

1）点开触发拍照按钮设置

2）通过添加按钮添加并设置相机接收信号命令

3）命令信号注解：

● 字段类型：可选择相机触发信号，流水号，可编辑信号

●字段名：字段类型的别名（多用于可编辑信号）

● 字段索引：设置接收信号的索引位置；

●字段值：接收到的信号的值。

●字段长度：从字段索引开始计算，往后加字段长度为止作为一个信号。

● 转码类型：将信号进行转码，可选择NONE(不转码)，ASCII码，UTF8,Unicode,Default，根据项目需求进行选择。

● 进制转换：可选十进制，十六进制，根据项目需求进行选择。

● 应用到：将本条信号命令设置到到指定一个相机中。可选择所有相机或独立相机

# 2.1.2.5 就绪清空 正在拍照 拍照完成信号（发送）

就绪清空正在拍照 拍照完成 信号为AOI向下位机发送指令的信号，可设置指令信号，流水号，可编辑信号

1）点开信号按钮进行设置

2）通过添加按钮添加指令

3）命令信号注解：

●字段名：字段类型的别名（多用于可编辑信号）

● 字段索引：设置发送信号的索引位置；

●字段值：发送信号的值。

●字段长度：从字段索引开始计算，往后加字段长度为止作为一个信号。

● 转码类型：将信号进行转码，可选择NONE(不转码)，ASCII码，UTF8,Unicode,Default，根据项目需求进行选择。

● 进制转换：可选十进制，十六进制，根据项目需求进行选择。

● 应用到：将本条信号命令设置到到指定一个相机中。可选择所有相机或独立相机

# 2.1.2.6 检测完成信号（发送）

检测完成信号为AOI检测完成之后向下位机发送的结果信号，可设置指令信号，流水号，质量信号，颜色信号，可编辑信号

1）点开检测完成按钮进行设置

2）通过添加按钮添加指令

3）命令信号注解：

● 字段类型：可选指令信号，流水号，质量信号，颜色信号，可编辑信号

●字段名：字段类型的别名（多用于可编辑信号）

● 字段索引：设置发送信号的索引位置；

●字段值：发送信号的值。

●字段长度：从字段索引开始计算，往后加字段长度为止作为一个信号。

● 转码类型：将信号进行转码，可选择NONE(不转码)，ASCII码，UTF8,Unicode,Default，根据项目需求进行选择。

● 进制转换：可选十进制，十六进制，根据项目需求进行选择。

● 应用到：将本条信号命令设置到到指定一个相机中。可选择所有相机或独立相机

4）质量信号，颜色信号比较特殊，信号值需要在质量设置界面和颜色设置界面单独进行设置

# 2.1.2.7 通讯连接测试

设置好IP和信号后可在设置界面进行测试

1）点击连接按钮，查看通讯能否正常连接。

2）如果连接失败，可以利用Ping功能查看网口是否已经连接

点击Ping按钮，等待接受数据窗口反馈的信息，如果数据包0丢失，说明网口硬件已连接

3）确认硬件连接之后可在软件界面接受数据，查看数据是否有误
