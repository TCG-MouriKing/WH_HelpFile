    # 第4章 常见问题与解答

    # 4.1编译错误

    按照编译提示查看流程是否配置完整：例如下面提示颜色流程错误，未找到名称Region2等，都属于流程节点未完成连接

    # 4.2模板库、预处理库不存在

    检查软件运行路径“CellGrid”文件夹中是否存在模板库与处理库，如果没有响应的库，需要拷贝一个进去。

    # 4.3存在多个表达式时报错

    表达式编辑完之后请先点按计算查看计算值

    # 4.4不小心误操作后，没有撤销功能（返回上一步）类似Ctrl+Z，是否能增设这个功能

    流程右上角有重置按钮恢复到最近保存的一步

    # 4.5图像灰度值数值不准，把鼠标指针多次放在同一个像素内，灰度值不一致，且误差很大

    像素右下角才为当前像素灰度值

    # 4.6流程无法使用Delete键删除

    方式一：流程删除快捷键：删除 Ctrl+del；方式二：在工具标题上右键菜单删除

    # 4.7有时第一次编译会提示文件出错，尝试多次编译后成功

    二次重新生成代码编译即可

    # 4.8连续检测图片，偶尔会报异常；随后重启软件，打开刚才的配方会提示报错，需要多尝试几次才能打开

    管理员身份启动

    # 4.9 打开配方报错

    这个错误为WH Vision.dll，被设置为只读的，把只读的勾选去掉
