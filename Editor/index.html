# 产品介绍

# 产品背景

在工业4.0、智能制造等战略带动下，物联网、5G、云计算以及人工智能等新一代信息技术正在加速与传统工业融合。其中机器视觉技术在传统工业转型的过程中起到极大的推动作用，该技术已在各行各业中被广泛应用。视觉软件主要由软件框架、与算法两大部分组成，目前行业中大部分视觉软件是以产线运行软件、算法调试集成一体的方式在应用。此方式需要具有较好视觉知识的工程师在现场才能完成调试、应用等工作，不方便调试与售后支持。

因此研发一款算法与应用分开的视觉架构，二次封装简化算法应用操作流程，提高产品的稳定、简易操作及功能易扩展性，方便现场技术支持与售后维护工作。

# 产品定位

软件包含机器视觉提取筛查等工具模块、流程编辑器、测试算法、导出算法库功能，用户只需将测试好的算法库导出，提供给AOI运行运行软件应用。主要应用于光伏行业硅片、电池片环节检测的一系列产品，包括硅片环节的尺寸检测、崩边检测、脏污检测、隐裂检测等模组，以及电池片的成品检测、丝网出板检测、PE镀膜检测、隐裂检测、PL检测等模组。该软件以制作导出算法为核心，可广泛适用于各行视觉检测需求中。
